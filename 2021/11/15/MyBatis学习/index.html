MyBatis

概念

  • 持久层
    • 负责将数据保存到数据库的那一层代码
    • JavaEE三层架构 : 表现层, 业务层, 持久层
  • 框架 :
    • 框架就是一个半成品软件. 是一套可重用的, 通用的, 代码基础代码模型
    • 在框架的基础上构建软件编写更加高效, 规范, 通用, 可扩展

Mapper代理开发

  • 使用Mapper代理方式完成入门案例
    1. 定义与SQL映射文件同名的Mapper接口, 并将接口与SQL映射文件放在同一目录下
    2. 设置SQL映射文件的namespace属性为Mapper接口全限定名
    3. 在Mapper接口中定义方法, 方法名为SQL映射文件中sql语句的id, 并保持参数类型与返回值类型一致
    4. 编码
      1. 通过SqlSession的getMapper方法获取Mapper接口的代理对象
      2. 调用对应方法完成sql的执行

  • 数据库表的字段名称 和 实体类的属性名称 不一样, 则不能自动封装数据

    • 解决方法 :
      1. 起别名
      2. sql片段
      3. resultMap :
        1. 定义resultMap标签
        2. 在<select>标签中, 使用resultMap属性 替换 resultType属性

  • resultMap用于封装结果集

  • 参数占位符

    • #{} : 会将其替换 为 ? , 可以防止sql注入
    • &{} : 拼sql, 会发生sql注入
      • 使用时机 :
        • 参数传递时 : #{}
        • 表名或列名不固定时 : &{}(很少用到)
    • 参数类型 : parameterType(可省略
    • sql语句特殊字符处理
      • 转义字符 : <
      • CDATA区 : <![CDATA[ < ]]> (打CD自动提示)

  • 参数接收

    1. 散装参数 : 需要使用@(Param)(“SQL中参数占位符的名称”)
    2. 实体类封装参数 :
      • 需要保证SQL中的参数名与实体类的参数名对应上
    3. map集合
      • 需要保证SQL中的参数名与map的键值对应上

  • 动态SQL : SQL语句会随着用户的输入或外部条件的变化而变化,

    • MyBatis 对动态SQL的支持
      • if
      • choose(when, otherwise)
      • trim(where, set)
      • foreach
    • if : 用于判断参数是否有值, 使用test属性进行条件判断
      • 存在的问题 : 第一个条件不需要逻辑运算符
        • 解决方法 :
          • 使用恒等式让所有条件格式都一样
          • <where>标签替换where关键字
  • 增删改需要
    • sqlSession.commit()来提交事务,
    • SqlSession sqlSession = sqlSessionFactory.openSession(true);来开启动态提交
  • 当传入数组时,
    • 使用@(Param)(“SQL中参数占位符的名称”)
    • 使用array
  • foreach属性
    • collection : 容器
    • item : 迭代元素
    • open : 开始符号
    • close : 结束符号
    • separator : 分隔符

批量删除示例代码

  • MyBatis参数传递(建议都使用@Param注解来修改Map集合中的默认键名)

    • 单个参数

      1. POJO类型 : 直接使用, 实体类属性名 和 参数占位符名称一致

      2. Map集合 : 直接使用, 键名 和 参数占位符名称一致

      3. Collection : 封装成Map集合

        map.put("collection", collection集合)

        map.put("arg0", collection集合)

      4. List : 封装成Map集合

        map.put("collection", list集合)

        map.put("list", list集合)

        map.put("arg0", list集合)

      5. Array : 封装成Map集合

        map.put("array", array数组)

        map.put("arg0", array数组)

      6. 其他类型 : 直接使用

    • 多个参数 : 封装成Map集合

      map.put("arg0", 参数值1)

      map.put("param1", 参数值1)

      map.put("arg1", 参数值2)

      map.put("param2", 参数值2)
