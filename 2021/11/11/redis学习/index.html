3.3 string 类型相关指令

命令 说明
set 设置一个 key/value
get 根据 key 获得对应的 value
mset 一次设置多个 key/value
mget 一次获得多个 key 的value
getset 获得原始 key 的值, 同时获得新值
strlen 获得对应 key 存储 value 的长度
append 为对应 key 的 value 追加内容
getrange 截取 value 的内容
setex 设置一个 key 存活的有效期(秒)
psetex 设置一个 key 存活的有效期(毫秒)
setnx key不存在时添加
msetnx 所有 key 都不存在时, 设置多个 key/value,
decr 对数值类型的value做 -1 操作
decrby 对数值类型的value做 减法 操作
incr 对数值类型的value做 +1 操作
incrby 对数值类型的value做 加法 操作
incrbyfloat 对数值类型的value做 加法 操作(支持浮点数)

3.4 List类型

命令 说明
lpush 将某个值加入到 key 列表头部
lpushx 同 lpush, 但是要保证 key 存在
rpush 将某个值加入到 key 列表末尾
rpushx 同 rpush, 但是要保证 key 存在
lpop 返回和移除列表左边的第一个元素
rpop 返回和移除列表右边的第一个元素
lrange 返回某个下标区间内的元素
llen 获取列表元素个数
lset 设置某个指定索引的值
lindex 获取某个指定索引位置的元素
lrem 去重
ltrim 保留列表中特定区间内的元素
linsert 在某个元素之前, 之后插入新元素

3.5 SET类型

命令 说明
sadd 为集合添加元素
smembers 显示集合中所有元素, 无序
scard 返回集合中元素个数
spop 随机返回一个元素, 并将该元素删除
smove 从一个集合中向另一个集合移动元素
srem 从集合中删除元素
sismember 判断一个集合中是否含有这个元素
srandmember 随机返回元素
sdiff 求补集
sinter 求交集
sumion 求并集

3.6 ZSET类型

命令 说明
zadd 添加一个有序集合元素
zcard 返回集合的元素个数
zrange升序, zervrange降序 返回一个范围内的元素
zrangebyscore 按照分数查找一个范围内的元素
zrank 返回排名
zrevrank 倒序排名
zscore 显示某个元素的分数
zrem 移除某个元素
zincrby 给某个元素加分

3.7 hash

命令 说明
hset 设置一个 key/value 对
hget 获得一个 key 对应的value
hgetall 获得所有的键值对
hdel 删除一个键值对
hexists 判断一个key是否存在
hkeys 获得所有的key
hvals 获得所有的value
hmset 设置多个键值对
hmget 获得多个键值对
hsetnx 当key不存在时, 设置一个键值对
hincrby 对value进行加法
hincrbyfloat 对value进行加法(支持浮点数)
