AJAX

介绍

  • 概念 : AJAX(Asynchronous JavaScript And XML) : 异步的的 JaxaScript 和 XML
  • 作用 :
    1. 与服务器进行数据交换 : 通过AJAX可以给服务器发送请求, 并获取服务器响应数据
      • 使用了AJAX和服务器通信, 就可以使用HTML + AJAX 来替换 JSP 页面
      • 异步的交互 : 可以在不重新加载整个页面的情况下, 与服务器交换数据并更新部分网页的技术, 如 ; 搜索联想, 用户名是否可用校验…

AJAX快速入门

  1. 编写AjaxServlet, 并使用response输出字符串
  2. 创建XMLHttpresponse对象 : 用于和服务器交换数据
  3. 向服务器发送请求
  4. 获取服务器响应数据

案例

  • AJAX验证用户名是否存在
    1. 给用户名输入框绑定光标失去焦点事件
    2. 发送ajax请求, 携带username参数
    3. 服务器端接受用户请求
    4. 调用service查询User
    5. 返回标记
    6. 浏览器端处理响应
